## ⚡ Quick Start

```bash
git clone https://github.com/Nabaikabaia/Batman-md.git
cd Batman-md
npm install
```

Edit settings.js - change ownerNumber to your WhatsApp number:

```js
ownerNumber: '2349049636843',  // Your number here
```

```bash
npm start
```

Enter the pairing code in WhatsApp → Settings → Linked Devices → Link a Device

---

📋 Basic Commands

Command Action
.menu Show all commands
.ping Check bot status
.play <song> Download music
.sticker Make sticker
.owner Contact owner

---

⚙️ Configuration

Edit settings.js:

```js
ownerNumber: '2349049636843',   // Your WhatsApp number
prefix: '.',                     // Command prefix
commandMode: 'public',           // public / private
version: '1.0.3'
```
